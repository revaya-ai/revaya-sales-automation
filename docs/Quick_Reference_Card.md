# Webhook Form Fields - Quick Reference Card

## Webhook 1: Lead Intake (Contact Form)
**URL:** `/webhook/revaya-lead-intake`

```
┌─────────────────────────────────────────────────┐
│  REQUIRED FIELDS (8)                            │
├─────────────────────────────────────────────────┤
│  □ first_name        "Sarah"                    │
│  □ last_name         "Johnson"                  │
│  □ email             "sarah@company.com"        │
│  □ phone             "555-123-4567"             │
│  □ company_name      "Acme Corp"                │
│  □ website           "https://acmecorp.com"     │
│  □ interested_in     "Website redesign"         │
│  □ additional_info   "Current site is..."       │
└─────────────────────────────────────────────────┘
```

**Triggers:**
- 6 AI agents research the lead
- Google Doc created with research summary
- Email sent to hello@revaya.com
- Data saved to Google Sheets

## Webhook 2: Post-Call Proposal (Internal Form)
**URL:** `/webhook/revaya-post-call`

```
┌─────────────────────────────────────────────────┐
│  REQUIRED FIELDS (10)                           │
├─────────────────────────────────────────────────┤
│  □ lead_id              "LEAD-20241129..."      │
│  □ services_needed      "Website, chatbot"      │
│  □ pain_points          "Slow website..."       │
│  □ business_goals       "Increase conv by 30%"  │
│  □ budget_discussed     "Yes" / "No"            │
│  □ budget_amount        "$5,000 - $8,000"       │
│  □ timeline_preference  "2-3 months"            │
│  □ priority_level       "High" / "Med" / "Low"  │
│  □ call_notes           "Decision maker on..."  │
│  □ call_date            "2024-11-29" (optional) │
└─────────────────────────────────────────────────┘
```

**Triggers:**
- Proposal generated by AI
- Saved to Google Drive
- Email sent with proposal link
- Google Sheets updated

## Common Mistakes

❌ **Wrong:** `firstName` (camelCase)
✓ **Right:** `first_name` (snake_case)

❌ **Wrong:** Missing `email` field
✓ **Right:** Email is required (used as unique ID)

❌ **Wrong:** Website without https://
✓ **Right:** Full URL: `https://domain.com`
